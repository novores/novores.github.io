✨ Pendahuluan

Emptty merupakan salah satu Display manager yang simple, emptty ditulis dalam bahasa Go. Emptty sendiri berbasis CLI, inilah yang membuatnya cukup simple untuk dioperasikan, namun ada beberapa konfigurasi yang harus kita tentukan terlebih dahulu, agar emptty dapat bekerja sesuai keinginan kita.

📦 Pemasangan

Sejauh ini, Untuk pengguna Archlinux emptty tersedia di AUR.

ketikan perintah berikut untuk menginstall emptty dari source file-nya pastikan kamu telah menginstall basic tools untuk build package. di Archlinux pastikan kamu telah menginstall paket base-devel.

$ git clone https://github.com/tvrzna/emptty.git
$ cd emptty
$ sudo make Makefile

⚙️ konfigurasi

  • pastikan emptty telah terinstall.

    $ emptty -v

  • sekarang kita akan mengedit file config emptty di /etc/emptty/conf.

    # TTY, dimana emptty dimulai.
TTY_NUMBER=2

# mengizinkan untuk beralih antar TTY.
SWITCH_TTY=true

# mengizinkan untuk membuat output dari /etc/issue di daemon mode.
PRINT_ISSUE=true

# namas user untuk ditentukan, jika AUTOLOGIN diizinkan, pengguna ini akan otomatis masuk.
DEFAULT_USER=noval

# mengizinkan Autologin, jika DEFAULT_USER telah ditentukan, dan bagian dari nopasswdlogin group.
AUTOLOGIN=false

# default session yang digunakan, jika Autologin diizinkan. jika session tidak ditemukan dalam daftar session, itu akan diproses dengan cara manual.
AUTOLOGIN_SESSION=bspwm

# bahasa default, jika pengguan belum mengaturnya dalam init script.
#LANG=en_US.UTF-8

# memulai desktop dengan memanggil "dbus-launch".
DBUS_LAUNCH=true

# memulai Xorg desktop dengan memulai "~/.xinitrc" skrip, jika bernilai true, file ada dan wm/de yang dipilih adalah sesi xorg, itu mengesampingkan peluncuran dbus.
XINITRC_LAUNCH=false

# Prints available WM/DE each on new line instead of printing on single line.
VERTICAL_SELECTION=false

  • dan terakhir, untuk mengaktifkan emptty saat booting, ketik perintah berikut di shell.

    systemctl enable emptty.service
Peringatan

perintah diatas hanya diperuntukan untuk pengguna systemd emptty telah menyediakan berbagai init script dalam source foldernya dan semuanya akan terinstal otomatis ketika kamu menginstall emptty melalui script Makefile. Kamu hanya perlu mengetikan perintah berikut untuk mengaktifkan emptty.

sudo rc-update add emptty
