Berbagi File Antar Device Dengan Wormhole William

  • 12/3/2022
  • 2 menit membaca

Deskripsi

wormhole-william adalah implementasi bahasa go dari magic-wormhole.

wormhole sendiri merupakan suatu command-line tool yang berisi perintah untuk mengirim dan menerima file, direktori, serta teks, secara aman. ini untuk mengirim dan menerima file dengan menggunakan wormhole code yang mudah dibaca sebagai “kunci pembuka” contoh kunci pembuka yang dimaksud:5-boy-farming.

wormhole juga menyediakan enkripsi end-to-end, titik akhirnya terhubung dengan 2 endpoints yang sama.

Kenapa menggunakan wormhole-william alih-alih wormhole?

karena wormhole-william memiliki beberapa fitur tambahan yang tidak ada di wormhole dasar, seperti dukungan qr code, built-in script untuk auto-completion di beberapa jenis shell, opsi flag yang bervariasi, dan dukungan QR code untuk handphone.

Instalasi

  • AUR sudo pacman -S wormhole-william
  • Go tool (memerlukan go terinstall) go install github.com/psanford/wormhole-william@latest

Penggunaan

untuk memudahkan pengetikan, buat alias untuk wormhole-william jadi wormhole.

lihat manpage untuk melihat cara penggunaanya secara lengkap.

  • Mengirim File

    $ wormhole send <path to file>

    • Mengirim File dengan QR Code

      $ wormhole send <path to file> --qr

      qr code memudahkan dalam memasukkan wormhole code di handphone. Dan ini termasuk fitur yang tidak ada di wormhole biasa.

  • Menerima file

    $ wormhole receive <kode wormhole dari pengirim>

  • Mengirim text

    $ wormhole send 
  Text to send: <string>

    atau

    $ wormhole send --text <string>
    • Mengirim text dengan QR Code 
      $ wormhole send --text <string> --qr

Selain mengirim file dan text wormhole juga dapat mengirim sebuah direktori/folder. Caranya sama dengan perintah untuk mengirim file. nantinya folder yang terkirim akan otomatis dikompres oleh wormhole dengan format zip

Mengirim dan menerima file di Android

Kamu juga dapat mengirim file dari dan ke Android dengan menggunakan wormhole-william.

aplikasi ini dapat kamu download di Play Store atau kamu dapat install langsung di github . Salah satu fitur menarik yang ada di aplikasi ini yaitu kita bisa mengscan QR Code yang dihasilkan dari wormhole, disini.
