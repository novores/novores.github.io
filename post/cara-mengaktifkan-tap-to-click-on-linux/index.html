Cara Mengaktifkan Tap-to-Click Di Archlinux

  • 15/09/2020
  • 1 menit membaca
!
Peringatan: postingan ini berumur lebih dari 365 hari. Informasi nya mungkin sudah kadaluwarsa .

Table of Contents

  • Intro
  • Requirements
  • Step by Step

Intro

Ketika menggunakan linux kita kadang-kadang tidak mendapati Sistem kerja komputer yang semestinya, kita dituntut untuk bersikap do it yourself.

maka dari itu kita harus benar-benar memahami cara kerja komputer yang sesungguhnya, dan dituntut mandiri menyelesaikan masalah yang ada. salah satu masalah yang pertama kali dirasakan ketika pertama kali mencoba linux adalah masalah tidak berfungsinya tap di touchpad laptop, saya akan mencoba memecahkan masalah ini secara runut agar mudah dipahami oleh anda.

Requirements

Install yang dibutuhkan: libinput

Step by Step

  1. Pertama pastikan file konfigurasi di /etc/X11/xorg.conf.d. Jika belum ada kita buat Symlink dari /usr/share/X11/xorg.conf.d/40-libinput.conf ke direktori /etc/X11/xorg.conf.d dengan mengetikkan perintah:

    sudo ln -s /usr/share/X11/xorg.conf.d/40-libinput.conf /etc/X11/xorg.conf.d/40-libinput.conf

  2. Dikutip dari Stack overflow ini adalah contoh config saya:

    Section "InputClass"
        Identifier "libinput touchpad catchall"
        MatchIsTouchpad "on"
        MatchDevicePath "/dev/input/event7"
        Driver "libinput"
        Option "Tapping" "on"          
        Option "ClickMethod" "clickfinger" 
        Option "TappingDrag" "on"
        Option "ScrollMethod" "twofinger"
        Option "DisableWhileTyping" "on"
        Option "TappingButtonMap" "lmr"
EndSection

copy lalu edit jika perlu. kunjungi libinput untuk mengetahui deskripsi secara mendetail.

Hati-hati ketika mengedit config file di folder root

  1. Reboot
