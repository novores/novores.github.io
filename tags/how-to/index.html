Mengautentikasi Git dengan SSH

Intro Mengautentikasi Git dengan SSH adalah dengan menghasilkan sepasang kunci enkripsi secara asimetris, dimana kunci disimpan di sistem dan yang satunya lagi di disimpan di server. dengan kata lain komputer tidak akan mendapat hak akses jika kunci yang dipasangkan salah atau berbeda, sehinga ini memungkinkan koneksi yang sangat-sangat aman dibanding kita menggunakan HTTPS. Dan keuntungan lainnya kita tidak lagi menggunakan token ketika akan mengatur repositori tapi cukup dengan password yang kita setel jika kamu menyetelnya nanti saat membuat atau meng-generate kunci SSH....
